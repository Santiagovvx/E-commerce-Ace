# E-commerce-Ace
E-commerce de accesibilidad WCAG 2.2 AA, basado en wireframes, UMB, Colombia - Jeienr Santiago Fernandez Vanegas
Descripción:

Este proyecto es un wireframe de un e-commerce de comida pensando en las personas con discapacidades, es una app accesible sobre un servicio de comida local que sea facil de usar para el usuario, con Botones grandes, con focus con hover con letras mas grandes, con colores con contrastes y muchas mas cosas, la app permite ver comidas, buscar productos, ver el detalle, agregarlos al carrito y hacer el pago, despues muestra una confirmación de compra.

Accesibilidad:

en el diseño se tuvieron en cuenta varias cosas de accesibilidad basadas en W3C y wcag 2.2:

botones grandes para que sean faciles de tocar
texto de 36px para mejor lectura
colores con buen contraste
estados como hover, focus y active
breadcrumbs para ubicarse mejor
mensajes de error claros en la busqueda
menu accesible desde cualquier pantalla

Notas:
este proyecto fue hecho como trabajo academico usando cosas de diseño de interfaces, accesibilidad y cree una interfaz sencilla pero que funcionara bien en celular sin tantas pantallas raras ni complicadas primero hice los wireframes en papel bien 
simples y luego los pase a Figma usando componentes para no repetir todo. puse las letras en 36px para que fuera mas facil la lectura de una persona con baja visibilidad y puse el separado 8px para la misma funcion y botones claros que se entiendan facil, tambien pense en que el usuario no se pierda, por eso puse botones de volver y una navegacion sencilla, agregue cosas como hover y focus para que uno sepa que esta tocando o recorriendo bien no tiene backend ni base de datos porque es solo un prototipo, osea es mas para mostrar como seria la app y como funcionaria, no es una app real todavia.
