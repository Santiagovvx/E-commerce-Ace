 Colores
Naranja: #FF4940
Blanco: #FFFFFF
Negro: #111
Gris claro: #D9D9D9
El naranja es el principal, se usa en botones y cosas importantes, el blanco se usa en barras y botones, el negro para texto y el gris par algunos botones
El contraste minimo es 4.5:1 para que todo se vea bien y no cueste leer

Tipografía
Tamaño minimo: 16px
use 32 porque me parecio adecuado para personas con dificultad de vision ademas de un espaciado de 8, para facilitar la lectura de textos y titulos pequenos

Espaciado 8px:
Me sirve para ordenar todo, separar textos botones y que no se vea todo pegado ademas de como ya lo mencione para facilitar la lectura de textos y titulos pequenos

Componentes:
Botones: Son grandes para que sean faciles de usar y puse varios como: botones con hover, con Focus, cuando pasas el mouse cambia de color, para menu, boton de rgreso o back que tienen sus diferentes acciones

Navegación:, Hay breadcrumbs para saber donde esta uno, tambien esta el menu en casi todas las pantallas, todo es simple para no perderse

Formularios: Nombre, Direccion, Telefono.
Botones: Confirmar, Volver
Mensajes: 
Error: cuando no hay resultados en la busqueda
comprar: mensaje de gracias por la compra
